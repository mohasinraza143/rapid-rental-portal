# rapid-rental-portal
Car Rental System is a web-based application developed to automate the car rental process. It allows users to register, login, browse available cars, and book cars online.

The system also includes an admin panel where administrators can manage cars, bookings, and users efficiently.

Features

User Registration and Login

Browse Available Cars

Car Booking System

Admin Dashboard

Manage Cars and Bookings

Database Management

Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Server: XAMPP / Apache

Purpose

The main purpose of this project is to simplify the car rental process and provide an efficient way to manage rental services online.
1. Download and Unzip the file on your local system copy carrental.
2. Put carrental folder inside the root directory
